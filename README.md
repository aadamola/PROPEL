# PROPEL

**AI-forward digital marketing for real estate professionals — Lagos, Nigeria.**

Propel is a remote-first digital marketing agency serving realtors, estate surveyors & valuers, real estate agents, and small-to-mid developers across Nigeria, starting in Lagos.
